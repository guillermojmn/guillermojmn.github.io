Embedded Orientation Monitoring & Alert System — ESP32 & IMU
Final Degree Project (TFG)
Sep – Dec 2025 +

Complete real-time IoT monitoring system capable of detecting dangerous orientations in an autonomous vehicle (AGV) and triggering local and remote alerts — even without network connectivity.

Key technical features
  • Microcontroller: ESP32 (WiFi + BT); inertial sensor: MPU-6500 (6-DOF)
  • Sensor fusion via complementary filter → roll, pitch & yaw estimation
  • Quaternion-based orientation (gimbal lock-free)
  • Local alarms: LED + buzzer active in degraded mode (no network required)
  • Remote alerts via Telegram Bot with real-time push notifications
  • Historical monitoring: InfluxDB + Grafana dashboards
Communication stack
  • I²C — ESP32 ↔ IMU
  • WiFi/TCP — ESP32 → Node.js backend (NDJSON streaming)
  • WebSocket — Backend → Three.js 3D web interface
  • HTTP — Backend → InfluxDB & Telegram Bot API
